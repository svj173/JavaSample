Динамический вывод графики - по мере поступления данных.

Окно фиксированного размера. Не растягивается. Т.е. старые данные исчезают с экрана.
График рисуется линией.

1) Демо 2
 - Одно окно.
 - Рисуются два графика - при нажатии на кнопки
     - рисовать толкьо первый график
     - рисовать толкьо второй график
     - рисовать оба одновременно
2) Демо 3
 - Три окна
 - Кнопки
     - рисовать только первый график
     - рисовать только второй график
     - рисовать только третий график
     - рисовать все одновременно
