        Проба отправки ява-обьектов.

    1. Простой поток
    Отправленный обьект сразу же принимается второй стороной.

    2. ZIP поток.
    Отправленные обьекты на другом конце принимаются толкьо по закрытию svj.zip-потока
первой стороной.
           gzipout.finish();
    Т.е. мы можем отправлять масив обьектов в течении получаса, а вторая сторона
будет все время в неведении.
    При падении сокета - данные теряются.

    EOFException генерится когда сокет закрывается и данных в потоке больше уже
не будет никогда.
